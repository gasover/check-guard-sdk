接口返回码说明

接入商每次调用接口时，可能获得正确或错误的返回码。开发者可以根据返回码信息调试接口，排查错误。

接口返回错误时包含以下JSON字符串（相关API可能会有附加数据）：

{"errCode":404,"errMsg":"资源未找到"}

接口返回正确时为实际数据，具体内容请参看相关API文档。

全局返回说明如下：

返回码 说明
-1 系统繁忙，请稍候再试
0 请求成功
400 请求参数错误
401 未认证
403 权限拒绝
404 资源未找到
500 服务内部错误
